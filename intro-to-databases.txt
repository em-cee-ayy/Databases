2. Explain when a database would be used. Explain when a text file would be used.

Databases are used to store large amounts of data that can be read and modified quickly. Databases are stored in binary format which is not readable by humans, so when it does need to be read by humans, a text file is used.

3. Describe one difference between SQL and other programming languages.

SQL is declarative rather than procedural, which allows us to be concerned with WHAT we want to find rather than how we will go about finding it.

4. In your own words, explain how the pieces of a database system fit together at a high level.

Databases are organized into tables that consist of columns and rows, where each row contains the data for of an entry and the columns define what the data represents.  The value in each cell can be queried.

5. Explain the meaning of table, row, column, and value.

Table: Structure that holds a collection of related data
Row: Horizontal group of values within a table
Column: Individual fields within a table
Value: Data contained in a cell
