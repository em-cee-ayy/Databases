1.
Adding: INSERT INTO <tablename> VALUES <your new values>
Updating: UPDATE <tablename> SET <values to be updated> WHERE <selector>
Deleting: DELETE FROM <tablename> WHERE <selector>

2. A person must clarify the table name, have the data that they want to add or update, and then have some sort of selector (WHERE id > 123 or WHERE id > 123 and price < 100). For deleting data, you only need the table name and selector.

3.
Boolean: Whether a person has registered their email or not
Integer: Regular number with no decimal or a userID
Money: The cost of an item
Date: A calendar date to keep track of their birthday to send discounts
